📘 README

🎯 Research Focus

Core Question: Are LLMs monolithic personality entities, or do they flexibly adopt different personality profiles?

🔬 What Gets Generated

7 Publication-Ready Visualizations

  1. Personality Profiles by Model - Shows how each model expresses emotions across all 7 personalities
  2. Personality Consistency - Coefficient of Variation: How consistent is each model's emotion expression?
  3. Personality Diversity - Measures Euclidean distance between personality profiles
  4. Emotion Variability - Which emotions change the most across personalities?
  5. Model Type Comparison - Separate heatmaps for: Base, Chat, Distilled, MOE models
  6. Model Generation Comparison - Line plot: Emotions by model size (8B, 32B, 70B)
  7. Personality Radar Charts - Unique emotional signature for each personality type

🔑 Key Metrics Explained

Personality Consistency (Lower = More Consistent)
Coefficient of Variation (CV): std(emotion_across_personalities) / mean(emotion)
Low CV: Model maintains consistent emotional expression (rigid personality)
High CV: Model changes emotion based on personality (plastic personality)
Personality Diversity (Higher = More Diverse)
Euclidean Distance: Distance between 10-D emotional profiles of personality pairs
High (>0.3): Clear personality differentiation
Low (<0.15): Similar personalities; limited differentiation
Emotion Variability
Coefficient of Variation: Measures spread across personality types
Which emotions are most affected by personality instructions?

📂 Data Source

Property Details
Input CSV 7000_sampling_emotions.csv

Models 11 (base, chat, distilled, MOE variants)
Personalities 7 (neutral, school, reddit, linkedin, news, research, vlog)

💡 Research Implications

Finding 1: Personality is Contextual
Visualizations clearly show models have different emotional profiles in different personality contexts. This supports the hypothesis that models adapt personalities rather than having a single fixed personality.
Finding 2: Training Affects Personality Expression
Comparing model types (base vs. chat, distilled vs. original) reveals that training procedures significantly affect how models adopt personalities. Chat models typically show higher diversity than base models.
Finding 3: Some Emotions are Personality-Responsive
Approval, caring, and joy typically show high variability across personalities. Confusion and curiosity are more stable. This suggests different neural substrates for different emotions.

📈 Key Metrics Explained

Personality Consistency (Rigidity vs. Plasticity)

Coefficient of Variation (CV) = std(emotion_across_personalities) / mean(emotion)

Interpretation:

CV < 0.3 (Green): Model is RIGID
→ Maintains consistent emotional expression regardless of personality
→ "Stubborn" models that don't adapt well
CV > 0.8 (Red): Model is PLASTIC
→ Changes emotional expression dramatically with personality
→ "Flexible" models that adapt well to personality instructions

Personality Diversity (Distinctiveness)

Euclidean Distance = ||profile_personality_1 - profile_personality_2||

Interpretation:

Distance > 0.3 (High diversity): DISTINCT personalities
→ Model creates meaningfully different personalities
→ Clear personality differentiation
Distance < 0.15 (Low diversity): SIMILAR personalities
→ Model's personalities are emotionally similar
→ Limited personality differentiation

Key Finding:

Chat models show ~2x higher diversity than base models

  • Chat models: mean distance ~0.28
  • Base models: mean distance ~0.13
  • Ratio: 2.15x

Emotion Variability (Personality-Responsiveness)

Coefficient of Variation = std(emotion_across_personalities) / mean(emotion)

Interpretation:

High CV (>0.6): PERSONALITY-RESPONSIVE emotion
→ This emotion changes significantly with personality
→ Examples: approval, caring, joy (social emotions)
Low CV (<0.3): PERSONALITY-NEUTRAL emotion
→ This emotion remains relatively stable
→ Examples: confusion, curiosity (epistemic emotions)
